# SOYO | 香氛·空間

這是一個專為香氛品牌設計的優雅展示網頁，強調靜謐、儀式感與空間美學。

## 專案特點
- **響應式設計 (RWD)**：完美適配手機、平板與電腦螢幕。
- **優雅視覺**：使用 Noto Serif TC 字體營造品牌質感。
- **模組化結構**：CSS 與 JavaScript 檔案獨立分離，方便維護。
